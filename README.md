# 双色球模拟选号器

一个简洁美观的双色球模拟选号工具，支持随机选号和幸运选号模式。

## 功能特点

- 🎲 **随机选号**：生成6个红球(1-33) + 1个蓝球(1-16)
- ✨ **幸运选号**：优先选择常见幸运数字
- 📋 **选号记录**：自动保存最近20条记录
- 📱 **响应式设计**：支持手机和电脑访问

## 在线访问

https://zhaorichy.github.io/ssq-picker/

## 技术栈

- HTML5
- CSS3 (渐变、动画、Flexbox)
- Vanilla JavaScript (ES6+)

## 本地运行

直接在浏览器中打开 `index.html` 即可。
